JVM (Wirtualna Maszyna Java) potrafi zająć całą dostępną pamięć RAM, dlatego warto z góry przydzielić jej maksymalną pamięć.
Służy do tego parametr -Xmx.

Przykładowo dla limitu 1024MB RAM należy użyć:
-Xmx1024M

Wartość Xmx powinna być mniejsza niż dostępny RAM, gdyż JVM potrafi zająć RAMu więcej niż zostanie przydzielone.
