Jak rozumieć zużycie RAMu?

Aplikacje - pamięć zużyta przez aplikacje aby te mogły działać, jest to główny powód dla posiadania RAMu :)

Cache - często odczytywane dane z dysku są przechowywane w pamięci aby system mógł je szybciej odczytywać.

Bufory - obszar do którego zbierane są dane do zapisu na dysk, może być duży jeśli dysk jest zbyt wolny.

Nieużywane - pamięć całkowicie nieużywana i gotowa do "zasiedlenia", według jądra Linux jest to zmarnowany RAM, lepiej gdy jest przeznaczony na cache.

System operacyjny sam dba o odpowiednie wartości pamięci cache oraz buforów. Nie jest wymagana interwencja użytkownika.
